# Испытание дня — прокачай себя

Навык для Алисы, который выдаёт пользователю ежедневные испытания по категориям:

* спорт
* интеллект
* учёба
* творчество
* случайное

После выполнения пользователь говорит: **выполнил**, и навык начисляет опыт, повышает уровень и считает серию дней.

## Что есть в проекте

* ветвление диалога;
* хранение состояния пользователя;
* случайный выбор категории и задания;
* мини-квесты;
* случайные события;
* XP, уровни и стрик;
* unit-тесты;
* линтер и code style fixer;
* git hooks для запуска проверок.

## Стек

* black


## Git hooks

После установки pre-commit хук будет запускать:

* ruff;
* black;
* pytest.

Команды:

```bash
pre-commit install
pre-commit run --all-files
```

## Как это работает

1. При первом запуске навык приветствует пользователя и просит выбрать категорию.
2. Пользователь называет категорию.
3. Навык создаёт испытание и сохраняет его в состоянии пользователя.
4. Пользователь говорит **выполнил**.
5. Навык начисляет XP, обновляет уровень и серию дней.

## Формула уровней

* 0–49 XP — уровень 1
* 50–99 XP — уровень 2
* 100–149 XP — уровень 3

И так далее: каждые 50 XP — новый уровень.
