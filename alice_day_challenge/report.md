# Отчёт по лабораторной работе
## Тема
Навык для Алисы: **«Испытание дня — прокачай себя»**

## Цель работы
Разработать голосовой навык для Алисы, который выдаёт пользователю задания по выбранной категории, хранит состояние профиля, начисляет опыт и поддерживает игровые механики.

## Задачи
1. Реализовать ветвление диалога.
2. Реализовать хранение состояния пользователя.
3. Добавить случайный выбор категории и задания.
4. Добавить систему опыта, уровней и серии дней.
5. Добавить мини-квесты и случайные события.
6. Подготовить unit-тесты, линтер, форматтер и git hooks.

## Описание логики
Пользователь запускает навык и видит предложение выбрать категорию:

- спорт
- интеллект
- учёба
- творчество
- случайное

После выбора категории навык формирует испытание.  
Если пользователь говорит «выполнил», система:

- проверяет, что активное испытание действительно есть;
- начисляет опыт;
- обновляет уровень;
- обновляет серию дней;
- очищает активное задание.

## Состояния пользователя
В проекте используются следующие данные:

- `xp` — общий опыт;
- `level` — текущий уровень;
- `streak` — серия дней подряд;
- `last_completed_date` — дата последнего выполнения;
- `challenge` — активное испытание.

## Игровые механики
### 1. XP и уровни
За выполнение испытания начисляется 10 XP.  
Если включено событие-бонус или хардкор, опыт удваивается.

Уровни считаются так:

- каждые 50 XP — новый уровень;
- уровень 1 соответствует 0–49 XP;
- уровень 2 — 50–99 XP.

### 2. Серия дней
Если пользователь выполняет задания несколько дней подряд, серия увеличивается.

### 3. Мини-квесты
Иногда вместо одного задания выдаётся цепочка из 3 шагов.

### 4. Случайные события
С вероятностью 10% включается бонус или хардкор-режим.
